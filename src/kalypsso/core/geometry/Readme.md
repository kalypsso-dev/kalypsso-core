# Purpose

This directory contains geometrical primitives aimed at helping a certain type of initial conditions implementation, based on constant hydrodynamics state inside space regions. Space regions can have a simple geometrical shape (rectangular box, sphere, triangle, tetrahedron, ...), or polygonal shape (a polyline in 2d, or a STL/PLY mesh in 3d).

A large part of the content of this directory is adapted from [ArborX](https://github.com/arborx/ArborX) licensed under BSD 3-clause.
